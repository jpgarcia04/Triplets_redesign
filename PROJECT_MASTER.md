# PROJECT MASTER — Triplets Atelier Redesign

## 1. Contexto del proyecto

Este proyecto consiste en rediseñar y reconstruir una web para Triplets Atelier, una florería con sede en Bosque Real, CDMX.

La marca ya cuenta con un sitio web funcional, pero el objetivo de este proyecto es crear una versión alternativa usando tecnologías web base: HTML, CSS y JavaScript.

La tarea permite libertad creativa en estructura, acomodo y experiencia de usuario, siempre respetando la esencia visual de la marca y su paleta de colores.

---

## 2. Objetivo general

Crear una web más limpia, clara, elegante y orientada a conversión.

La prioridad es que, al entrar a la página, el usuario vea casi todo lo importante sin tener que navegar demasiado:

- Qué es Triplets Atelier.
- Qué servicios ofrece.
- Dónde se encuentra.
- Cómo pedir o cotizar.
- Por qué la marca se siente premium y diferente.

---

## 3. Objetivos específicos

### UX y conversión

- Reducir fricción para contactar.
- Hacer visible el CTA principal desde el primer pantallazo.
- Separar claramente los servicios principales.
- Dar prioridad a WhatsApp/contacto cuando aplique.
- Evitar que el usuario tenga que recorrer toda la web para entender la oferta.

### Diseño visual

- Mantener una estética limpia, suave y premium.
- Respetar la paleta de colores de la marca.
- Usar espacios amplios, tipografías elegantes y animaciones sutiles.
- Evitar saturación visual.
- Construir una experiencia “smooth”, especialmente en móvil.

### Código

- Usar HTML semántico.
- Separar CSS y JavaScript correctamente.
- Mantener estructura ordenada de archivos.
- Preparar el proyecto para crecer por secciones.
- Documentar decisiones importantes en GitHub.

---

## 4. Servicios principales a mostrar

### Círculo Blossom

Servicio de suscripción floral para hogares u oficinas.

Debe responder claramente:

- Qué incluye.
- Cada cuánto llega.
- Para quién es.
- Cómo contratar.
- Qué zona cubre.

### Blossom Bar

Servicio floral para eventos.

Debe responder claramente:

- Qué tipo de eventos cubre.
- Cómo funciona.
- Qué valor aporta.
- Cómo se cotiza.

### Atelier / Kiosco

Experiencia física en Bosque Real.

Debe responder claramente:

- Dónde está.
- Qué puede hacer el cliente ahí.
- Horarios.
- Cómo llegar/contactar.

---

## 5. Principio principal de diseño

La home debe funcionar como una landing page.

No debe depender de que el usuario navegue por muchas páginas para entender el negocio.

La primera pantalla debe comunicar:

1. Marca.
2. Ubicación.
3. Propuesta de valor.
4. Servicios.
5. Botón de contacto o acción.

---

## 6. Estructura propuesta de la home

1. Header limpio.
2. Hero con mensaje claro y CTA.
3. Cards de servicios principales.
4. Sección “Cómo funciona”.
5. Planes o formas de compra.
6. Galería visual.
7. Sección de sostenibilidad/diferenciador.
8. Ubicación y contacto.
9. Footer.

---

## 7. Páginas futuras

Para el MVP inicial se trabajará primero en `index.html`.

Después se pueden crear:

- `suscripcion.html`
- `eventos.html`
- `atelier.html`
- `contacto.html`
- `galeria.html`

---

## 8. Paleta visual

Pendiente de ajustar con los colores exactos de la marca.

De momento se usará una base premium y floral:

- Fondo cálido claro.
- Texto marrón oscuro.
- Acentos tierra/terracota.
- Verde suave.
- Rosa floral muy sutil.

La paleta real debe validarse visualmente contra la web original o materiales oficiales.

---

## 9. Reglas del proyecto

- No copiar código de la web actual.
- No usar imágenes, logos o recursos privados sin permiso.
- No clonar la web tal cual: mejorar estructura y experiencia.
- Priorizar mobile-first.
- Documentar cambios importantes.
- Mantener commits claros.
