# Triplets Atelier Redesign

Rediseño conceptual y técnico de la web de Triplets Atelier, una florería con sede en Bosque Real, CDMX.

El objetivo no es copiar la web actual tal cual, sino construir una versión más clara, limpia, responsive y orientada a conversión usando HTML, CSS y JavaScript.

## Objetivo principal

Crear una web premium, smooth y fácil de navegar donde el usuario entienda rápidamente:

- Qué vende la marca.
- Dónde está ubicada.
- Qué servicios ofrece.
- Cómo comprar, cotizar o contactar.
- Qué diferencia a Triplets Atelier de una florería genérica.

## Stack inicial

- HTML5 semántico.
- CSS3 con variables, responsive design y animaciones suaves.
- JavaScript vanilla para interacciones básicas.
- Git/GitHub para control de versiones y evidencias.
